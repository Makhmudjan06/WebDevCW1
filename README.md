# WebDevCW1
Project Title: Evklid Website

Description:

This is a static HTML, CSS, and JavaScript website for Evklid, a design company. It was developed as a coursework project for web development. The website provides information about the company's services and includes a contact form for potential clients to submit inquiries.

Key Features:

Clean and Responsive Design: The website is designed to be visually appealing and easily accessible on various devices.
Informative Content: Provides detailed information about Evklid's services and mission.
Functional Contact Form: Allows users to submit inquiries directly to Evklid.
Used:HTML, CSS, JavaScript
